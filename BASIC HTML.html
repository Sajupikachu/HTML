BASIC HTML

HTML ELEMENTS

  • PARAGRAPH TAG
    -Paragraph tag is container tag in html

  • BODY TAG
    -Body tag defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.

  • TITLE TAG
    -Title tag specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab)

  • H1 TAG
    -h1 tag defines a large heading

  • HEAD TAG
    -Head tag contains meta information about the HTML page

  • HTML TAG
    -HTML tag is the root element of an HTML page

  • DOCTYPE TAG
    -Doctype tag declaration defines that this document is an HTML5 document

  • BIG TAG
    -Big tag makes text biggger

  • SMALL TAG
    -Small tag makes text smaller

  • BOLD TAG
    -Bold tag makes text look bolder

  • SUBSCRIPT TAG
    -Subscript tag used to make text as subscript

  • SUPERSCRIPT TAG
    -Superscript tag used to make text as superscript

  • COMMENT TAG
    -Comment tag used to comment a line

  • TABLE TAG
    -Table tag used to create a table

  • LIST TAG
    -List tag used to create a list
